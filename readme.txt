Use Widgets via ShortCodes

== Description ==

X2 Widget ShortCodes is a very small Plugin and gives you the possibility to use your Widgets as Shortcodes.

It creates 5 new Widget Areas ( shortcode 1 to shortcode 5 ).

The Shortcode is [x2_widget id=""] and comes with 2 parameter:

needed:
id -> the widget id 1 - 5 <br>

optional:
css  -> add your custom css

Just add the Shortcode into your Pages, Post.. that's it.

The plugin is designed for the x2 Theme but should also work with other themes.

== Installation ==

You can download and install X2 Widget ShortCodes using the built in WordPress plugin installer. If you download X2 Widget ShortCodes manually,
make sure it is uploaded to "/wp-content/plugins/".

Activate X2 Widget ShortCodes in the "Plugins" admin panel using the "Activate" link. If you're using WordPress Multisite, you can optionally activate X2 Widget ShortCodes Network Wide.
